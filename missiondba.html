🎮 Mission DBA

Un jeu interactif hébergé sur GitHub Pages — deviens Administratrice de Bases de Données !

Bienvenue dans Mission DBA ! 🐘

Dans ce jeu, tu te mets dans la peau d'une DBA (Administratrice de Bases de Données) qui prend en main un serveur tout neuf. Ta mission : le transformer en vraie base de données fonctionnelle, en tapant les commandes dans un terminal simulé — exactement comme dans la vraie vie !

👥 Jouable à plusieurs ! Ce jeu est une simple page web sur GitHub : chacun peut l'ouvrir sur son propre ordinateur ou téléphone avec le même lien. Pas d'installation, pas de compte — tout le monde joue en même temps, chacun à son rythme.

Voici les actions que tu vas réaliser :

  1. ⚙️ Installer le moteur de base de données (PostgreSQL)
  2. 🔌 Te connecter au moteur avec l'outil psql
  3. 🗄️ Créer la base de données du lycée
  4. 🔐 Créer un utilisateur et lui donner ses droits d'accès
  5. 🔍 Requêter les données : lire, insérer et supprimer
  6. 💾 Sauvegarder la base de données
  7. 🤖 Mission bonus : écrire toi-même un script avec l'éditeur vi !
  8. 📚 Grand final : écrire un playbook Ansible pour installer PostgreSQL sur tout un parc de serveurs !
⭐ Le score : chaque mission rapporte 100 points. Chaque erreur coûte 10 points, chaque indice 20 points (minimum 30 points par mission). Atteins 90 % du score maximum pour le rang « DBA Experte » ! 🏆

💡 Pas de panique : pour chaque mission, la commande à taper est affichée en clair et chaque mot de la commande est expliqué. Le but n'est pas de deviner, mais de comprendre ce que tu fais !
